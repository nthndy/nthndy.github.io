---

### What it is

A static portfolio site built with vanilla HTML, CSS, and JavaScript.  
No frameworks. No build step. Just files.

It features embedded interactive figures from [macrohet_worldwide](https://nthndy.github.io/macrohet_worldwide/) —  
a live preprint manuscript on *M. tuberculosis* infection heterogeneity in iPSC-derived macrophages.

---

### Stack

- HTML / CSS / JS
- [Plotly.js](https://plotly.com/javascript/) for interactive figures
- Hosted via GitHub Pages
